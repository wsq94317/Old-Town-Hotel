1. Room Assignment Design Principles

* 房型是硬约束
* 偏好是软约束
* 房间存在取舍型特征
* 清洁度 / 老旧度走暗线结算

2. Room Type Rules

* 大床 / 双床 / 家庭房
* 合法分房规则

3. Preference Rules

* 高楼层 / 低楼层
* 靠电梯 / 远离电梯
* 临街 / 背街
* 1~2 个随机偏好
* 可满足可不满足，不是硬要求

4. Early Check-in Design

* checkout 10am
* check-in 3pm
* 3pm 前入住算提前入住
* 可收费
* 是机会项，不是必须项

5. Late Checkout Design

* 请求通知
* 快速决策
* 批准到多久
* 收费梯度
* 风险来自压缩清洁和查房窗口

6. Visible vs Hidden Information

* 明线：房型 / 偏好 / 状态 / readiness
* 暗线：清洁感知 / 老旧感知 / 景观 / 噪音修正

7. Future Expansion Notes

* 强偏好 / 一般偏好
* upgrade / compensation
* 更复杂房型和位置系统
